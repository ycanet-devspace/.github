🌐 Personal Portfolio

Bienvenido a mi portafolio personal. Este proyecto fue creado para presentar mis habilidades, experiencia y proyectos más destacados en desarrollo de software.

🚀 Sobre el portafolio

Este portafolio funciona como una vitrina interactiva donde muestro proyectos construidos con diferentes stacks y tecnologías modernas.
El diseño se enfoca en ser simple, adaptable y profesional, pensado tanto para reclutadores como para colaboradores y clientes.

🛠️ Tecnologías utilizadas

Frontend: Angular / Next.js / React

Backend: NestJS / Node.js

Base de datos: PostgreSQL, Supabase

Integraciones: APIs externas, automatizaciones

📂 Contenido

Introducción personal – breve presentación y biografía técnica.

Proyectos destacados – listado con descripción, tecnologías y enlace al código o demo.

Contacto – enlaces a mis redes profesionales para conectar fácilmente.

🎯 Objetivo

Centralizar en un solo espacio mis proyectos e integraciones, mostrando tanto el código como el impacto de cada solución.
